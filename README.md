# Tetris Game

## Overview
A simple Tetris clone built with Pygame.

## Requirements
- Python 3.x
- Pygame (`pip install pygame`)

## How to Run
```sh
python tetris.py
```

## Controls
- **Left Arrow:** Move left
- **Right Arrow:** Move right
- **Down Arrow:** Soft drop
- **Up Arrow:** Rotate
- **Space:** Hard drop
- **C:** Hold piece

## Running Tests
```sh
python -m unittest tests.py
```
